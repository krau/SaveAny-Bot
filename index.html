        Save Any Bot #

        把 Telegram 上的文件转存到多种存储端.

        🎯 特性 #

        • 支持文档/视频/图片/贴纸…甚至还有 Telegraph
        • 破解禁止保存的文件
        • 批量下载
        • 流式传输
        • 多用户使用
        • 基于存储规则的自动整理
        • 监听并自动转存指定聊天的消息, 支持过滤
        • 使用 js 编写解析器插件以转存任意网站的文件
        • 存储端支持:
          • Alist
          • S3 (MinioSDK)
          • WebDAV
          • 本地磁盘
          • Telegram (重传回指定聊天)
