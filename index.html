        Save Any Bot #

        把 Telegram 上的文件转存到多种存储端.

        特性 #

        • 支持文档/视频/图片/贴纸… 甚至还有 Telegraph
        • 破解禁止保存的文件
        • 批量下载
        • 流式传输
        • 多用户
        • 基于存储规则的自动整理
        • 支持多种存储端:
          • Alist
          • Minio (S3 兼容)
          • WebDAV
          • Telegram (重传回指定聊天)
          • 本地磁盘
